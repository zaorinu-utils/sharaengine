# Shara Engine — Visão Geral

Shara Engine é um motor de jogos 2D em desenvolvimento, com foco em performance, modularidade e
integração com Lua para permitir criação rápida de jogos.

## Objetivos

- Arquitetura de motores de jogo
- Integração e segurança de linguagens de scripting
- Sistemas de renderização e abstrações gráficas
- Programação de baixo nível com Rust
- Organização, modularidade e boas práticas de engenharia de software
- Ferramentas e pipelines de build/asset
- Design de APIs simples e extensíveis
